Experiment(1)：Web Page

Posted on 2022-09-17 Edited on 2022-09-19 Views:

本文用于记录计网第一次实验——个人网页的步骤与过程。

Basic Information ｜ 基本信息

  • Device: MacBook Pro
  • System: macOS Monterey 12.4
  • Virtual Device: Windows 10 (Supported by Parallels 17)

Steps | 操作步骤

Preparation

JDK
  • Software: Java 2 SDK x.x.x
  • Links:
  • Path: C:2sdk

Configuration:

  1. 右键“此电脑”，点击“属性”-“高级系统设置”-“环境变量”
  2. 新建变量JAVA_HOME，添加变量值C:\JAVA\j2sdk
  3. 新建变量CLASSPATH，添加变量值.;%JAVA_HOME%\lib\tools.jar;%JAVA_HOME%\jre\lib\dt.jar; 其中.表示当前路径，;为windows系统分隔符
  4. 在系统变量PATH中点击“编辑”，然后点击“新建”添加变量值%JAVA_HOME%\bin
  5. 在cmd中输入java -version，java，javac可以检验是否安装配置成功
Tomcat
  • Software: Jakatar-tomcat-5.0.28（原先不太懂然后跟着助教传的十几年前的教程下了一个古早版本…但也还能用）
  • Links:
  • Path: C:（路径名不要加空格！！）

Configuration:

  1. 右键“此电脑”，点击“属性”-“高级系统设置”-“环境变量”
  2. 新建变量CATALINA_HOME，添加变量值C:\JAVA\Tomcat5.0
  3. 在系统变量PATH中点击“编辑”，添加变量值%CATALINA_HOME%\bin，%CATALINA_HOME%\lib，%CATALINA_HOME%\lib\servlet-api.jar

Run:

  • 安装目录下，进入\bin文件夹，点击startup.bat批处理程序使Tomcat运行
  • 浏览器中输入地址http://localhost.8080，看到Tomcat欢迎界面说明配置成功
  • 点击\bin中的shutdown.bat批处理文件停止运行
